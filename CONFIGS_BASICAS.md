# Python e Django
Certifique-se de ter instalados o python (linguagem de programação) e o django (framework).

### Instalação do Python
-- Linux (Ubuntu, Xubuntu, Debian, Fedora): não precisa, já vem instalada a versão 2 e 3.

-- Windows: baixar no site oficial (python.org) https://www.python.org/ftp/python/3.4.4/python-3.4.4.amd64.msi e ao instalar marque a opção -> ADD PYTHON AO PATH DO WINDOWS.

### Instalação do Django
-- Linux e Windows: abra um terminal e digite: 

`pip install django`
